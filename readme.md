# Macro Nutrient Prediction from Food Images

## Requirements
- Linux machine
- Git
- Python version 3.14 
- [UV](https://docs.astral.sh/uv/getting-started/installation/) Python package manager and build system


## Run the project
```bash
    git clone https://github.com/LuckyToaster/end_of_degree_project/
    cd end_of_degree_project
    python -m venv env
    python env/bin/activate
    pip install -r requirements.txt
    python -m src.scripts.load_data
    python -m pytest -v
```
